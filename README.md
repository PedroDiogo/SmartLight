SmartLight
==========

Automatic System to Control Lights

Requirements
------------

* Python 2
* SimpleCV 
* Python Curses
* Python JSON

(also if you want to enable the stream server)

* NodeJS
* SocketIO

Running SmartLight
------------------

**To run:**

    python motion.py

**To see the options:**

    python motion.py --help

**To run the stream server:**

    node server.js

**And open the client on your browser:**

    open status/index.html
